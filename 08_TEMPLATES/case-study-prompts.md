# Case Study Prompts (Complex Scenarios)

1) Microservices Incident (P1)
- Prompt: triage logs, isolate faulty service, propose rollback
- Output: mitigation steps + postmortem template

2) Legacy Migration
- Prompt: extract module into new package, add CI, deprecate old endpoints
- Output: PRs with small diffs + migration guide

3) CI/CD Hardening
- Prompt: add gates (lint, tests, security) and approval steps
- Output: updated workflows + validator job
